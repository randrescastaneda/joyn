# Conduct all unmatched keys checks and return error if necessary

Conduct all unmatched keys checks and return error if necessary

## Usage

``` r
check_unmatched_keys(x, y, out, by, jn_type)
```

## Arguments

- x:

  left table

- y:

  right table

- out:

  output from join

- by:

  character vector of keys that x and y are joined by

- jn_type:

  character: "left", "right", or "inner"

## Value

error message
