# joyn: Tool for Diagnosis of Tables Joins and Complementary Join Features

Tool for diagnosing table joins. It combines the speed of \`collapse\`
and \`data.table\`, the flexibility of \`dplyr\`, and the diagnosis and
features of the \`merge\` command in \`Stata\`.
